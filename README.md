# J.E.R.K. x Zenith | Prompt Engineering Lab & Audit Portfolio

This repository contains the complete submission for the **Prompt Engineering Internship** opening at **Baby Dino / Zenith India**. 

It comprises a comprehensive analytical audit report and a high-fidelity interactive companion web application demonstrating dynamic prompt engineering pipelines.

---

## 🚀 Project Overview

The project is structured around the intersection of **content, code, and culture** to audit and build solutions for two core in-house products: **Color Analysis AI** and **Zenith Fashion AI**.

### 1. Written Portfolio (`zenith_review.md`)
A polished, de-speculated written evaluation structured into four distinct parts:
* **About Me Cover**: Summary of background, programming skills, and AI tool stack.
* **Executive Summary**: One-page overview of reviewed portals, testing methods, and core findings.
* **Part 1: Product & Brand Audit**: Direct front-end testing observations (including camera exposure/lighting sensitivity scans) and recommendations.
* **Part 2: 20-Prompt Library**: Structured prompt templates designed for styling engines, social media copy, SEO content, and customer operations.
* **Part 3: Sample Work Iterations**: A case study walking through prompt iteration cycles (V1.0 vs V2.0) across three distinct demographic user profiles.
* **Part 4: Conclusion & Recommendations**: Suggestions for pre-analysis guides, pixel-to-text token parsers, and sizing schema enforcements.

### 2. Interactive companion App (Prompt Lab)
An interactive single-page application designed with a premium glassmorphic dark-neon theme (optimized for Gen Z style authority). Features include:
* **Audit Dashboard**: Displays J.E.R.K.'s workflow architecture, SWOT analysis, and recommendations.
* **Color AI Playground**: Builds dynamic prompt templates based on user drop-down inputs, simulating structured JSON styling outputs, latency, tokens, and seasonal palettes.
* **Fashion AI Playground**: Simulates outfit recommendations by pairing user profiles (body shape, occasion, vibes) with matching clothing catalogs and generated stylist descriptions.
* **20+ Prompt Portfolio**: A categorized list of copy-pastable prompts filtered by business function (Styling, Marketing, Brand/SEO, Operations).
* **Sample Work Tab**: Step-by-step comparative view of initial vs optimized prompts.

## 🛠️ How to Run Locally

Since the application is built using standard web technologies (HTML5, Vanilla CSS, and JavaScript), it runs entirely in the browser without any build tools.

### Option 1: Python HTTP Server (Recommended)
Navigate to the project root directory and run:
```powershell
python -m http.server 8000
```
Then, open your browser and navigate to:
👉 **[http://localhost:8000](http://localhost:8000)**

### Option 2: Node.js HTTP Server
Alternatively, you can use `http-server` from npm:
```bash
npx http-server -p 8000
```

### Option 3: Direct File Execution
You can also open `index.html` directly in any modern browser by double-clicking it.
